# 🎆 PyQt5 Particle Generator

A powerful and customizable particle simulation GUI built with **PyQt5**. This app lets you create stunning particle animations, tweak physical properties in real-time, and export the results as PNG images or animated GIFs! 🚀

## 🎛 Features

- 🎨 **Particle customization**: control gravity, size, velocity, fade, and life
- 🌈 **Color modes**: multicolor, single color, and white override
- 🖼️ **Background color** customization
- ▶️ **Start/Stop** particle simulation in real-time
- 💾 **Save** current canvas as PNG
- 📁 **Save/load** parameters in JSON
- 📽️ **GIF recorder**: record a 5-second animation (~60 FPS)

---

## 🧑‍💻 Installation

### Requirements

Ensure you have Python 3 and install dependencies:

```bash
pip install PyQt5 imageio numpy
```

### Run the App

```bash
python your_script.py
```

---

## ⚙️ Usage

- Adjust particle parameters via sliders and spinboxes
- Choose a custom color, or enable multicolor mode
- Click **Start Simulation** to generate particles
- Save results as PNG or record a GIF
- Store your favorite configurations using the **Save Parameters** button

## 🧰 Tech Stack

- **Python 3**

- **ImageIO** – export GIFs
